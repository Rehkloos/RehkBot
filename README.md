You need a `.env` to store your token which is formatted like this:

```
TOKEN=<bot token here>
PORT=8000
ROLE=<add ROLE ID>
PREFIX=<add a prefix for commands i.e !>
QUEUE=<add ID for queue channel>
```

then run `npm start` in directory
